# Welcome!

This is a simple [npm](http://npmjs.org) package for [RightJS](http://rightjs.org)

## Installation

    npm install rightjs

## Usage

    var RightJS = require('rightjs');

    var Klass = new RightJS.Class({
      initialize: function(param) {
        this.param = param;
      }
    });

    var klass = new Klass('param');
